UJICOBA PERCEPTRON (AND)

Dataset logika AND dengan update bobot sederhana.

X = np.array([[0,0],[0,1],[1,0],[1,1]]) y = np.array([0,0,0,1]) w = np.random.randn(2) lr = 0.1 for epoch in range(20): for i in range(len(X)): z = np.dot(X[i], w) output = 1 if z > 0 else 0 error = y[i] - output w += lr * error * X[i]

Target: Output harus 0, 0, 0, 1.

XOR & FUNGSI AKTIVASI

Fungsi Sigmoid:

def sigmoid(z): return 1 / (1 + np.exp(-z)) def sigmoid_deriv(a): return a * (1 - a)

Dataset XOR:

X = [[0,0],[0,1],[1,0],[1,1]] y = [[0],[1],[1],[0]]

KONFIGURASI BACKPROPAGATION

Inisialisasi Parameter Jaringan:

  • Input Neuron: 2
  • Hidden Neuron: 4
  • Output Neuron: 1
  • Learning Rate: 0.5
np.random.seed(1) W1 = np.random.randn(2, 4) W2 = np.random.randn(4, 1)

ALUR TRAINING

# Forward Pass Z1 = np.dot(X, W1); A1 = sigmoid(Z1) Z2 = np.dot(A1, W2); A2 = sigmoid(Z2) loss = np.mean((y - A2)**2) # Backward Pass dA2 = y - A2 dZ2 = dA2 * sigmoid_deriv(A2) dW2 = np.dot(A1.T, dZ2) dA1 = np.dot(dZ2, W2.T) dZ1 = dA1 * sigmoid_deriv(A1) dW1 = np.dot(X.T, dZ1) # Update Bobot W2 += lr * dW2; W1 += lr * dW1
