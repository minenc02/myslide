Definisi Hukum Siber

Cyber Law adalah aturan hukum yang mengatur penggunaan teknologi informasi, sistem elektronik, jaringan komputer serta kejahatan yang terjadi di ruang digital.

Tujuan Utama:

  • Melindungi sistem elektronik
  • Melindungi data
  • Menindak kejahatan siber
  • Menjamin keamanan informasi nasional

Dasar Hukum Siber di Indonesia

  • 📜 UU No. 11 Tahun 2008 tentang ITE
  • 📜 UU No. 19 Tahun 2016 (Perubahan ITE)
  • 📜 UU No. 1 Tahun 2024 (Perubahan Kedua ITE)
  • 📜 KUHP Nasional UU No. 1 Tahun 2023
  • 📜 UU Perlindungan Data Pribadi No. 27 Tahun 2022

Regulasi ini menjadi dasar utama penegakan hukum cybercrime di Indonesia.

Perbandingan Regulasi

📋 Regulasi Lama

  • KUHP lama (Wetboek van Strafrecht)
  • UU ITE 2008
  • Revisi ITE 2016

✨ Regulasi Baru

  • KUHP Nasional 2023
  • Revisi kedua UU ITE 2024
  • UU Perlindungan Data Pribadi 2022

Pentingnya Hukum Siber

1. 🛡️ Mencegah Korupsi

Mengurangi praktik korupsi melalui regulasi digital yang transparan dan terukur.

2. 🏛️ Keamanan Nasional

Meningkatkan pertahanan dari serangan teroris dan ancaman kejahatan siber.

3. 💼 Proteksi Bisnis

Melindungi dunia usaha dari penipuan, kerusakan reputasi, dan risiko hukum.

4. ⚖️ Penegak Hukum

Sarana untuk menghukum pelaku kejahatan teknologi informasi secara adil.

5. 📜 Bukti Elektronik Sah

Meningkatkan peluang diakuinya catatan elektronik sebagai alat bukti yang sah di pengadilan untuk kasus kejahatan biasa dan kejahatan komputer.

Jenis Serangan Keamanan

👂️ Interception

Pihak tidak berwenang mendapatkan akses informasi.

⏹️ Interruption

Penyerang menguasai sistem sebagian.

🎭 Fabrication

Menyisipkan objek palsu ke dalam sistem.

✏️ Modification

Merusak dan mengubah sistem secara keseluruhan.

Firewall

🔥 Definisi

Sistem/perangkat yang mengizinkan lalu lintas jaringan aman dan mencegah yang tidak aman.


⚙️ Fungsi Utama

  • Mengatur lalu lintas
  • Autentikasi akses
  • Melindungi sumber daya
  • Mencatat kejadian

IDS dan IPS

👁️ IDS (Intrusion Detection System)

Definisi: Sistem yang memantau jaringan untuk mendeteksi aktivitas mencurigakan.

  • Sifat: Pasif (Monitoring)
  • Aksi: Memberikan peringatan (Alert) saja
  • Peran: "Pengawas Keamanan"

🛡️ IPS (Intrusion Prevention System)

Definisi: Sistem keamanan yang mendeteksi dan secara aktif mencegah ancaman.

  • Sifat: Aktif (Blocking)
  • Aksi: Menolak atau memblokir paket berbahaya
  • Peran: "Penghalang Keamanan"

Jenis Firewall

1. 🖥️ Personal Firewall

Contoh: Microsoft Windows Firewall

2. 🌐 Network Firewall

Contoh: ISA Server, Cisco PIX

3. 🔢 IP Filtering

Bekerja pada level paket

4. 🔄 Proxy Server

Duplikasi komunikasi

Arsitektur Firewall (Bagian 1)

🖧 Dual-Homed Host

  • Dua Network Interface
  • Interface 1: Jaringan Internal
  • Interface 2: Internet
  • Routing Dinon-aktifkan (Mencegah koneksi langsung)

🏰 Bastion Host

Titik pertahanan utama firewall

Arsitektur Firewall (Bagian 2)

🛡️ Screened Host

  • Packet Filtering (Pengamanan utama via router)
  • Bastion Host Internal (Hanya dia yang koneksi ke Internet)
  • Akses Terkendali (Hanya tipe koneksi tertentu)

🔒 Screened Subnet

  • Layer Pengaman Tambahan (Perimeter)
  • Dua Screening Router
  • Keamanan Maksimal (Harus tembus 2 router)

Hacking vs Ethical Hacking

👤 Hacking

  • Menembus keamanan sistem komputer/jaringan
  • Mendapatkan akses tidak sah
  • Membahayakan sistem/mencuri info
  • ❌ Tanpa izin

🛡️ Ethical Hacking

  • Hacking atas izin pemilik
  • Menemukan kelemahan (pengujian)
  • Meningkatkan keamanan
  • ✅ Legal dan etis

Jenis Hacker (Tujuan)

🎩 White Hats

Mencari celah untuk analisa dan memperkuat keamanan.

🎭 Black Hats

Menggunakan kemampuan untuk kejahatan (crackers).

🎨 Grey Hats

Kadang white hat, kadang black hat (keuntungan).

💀 Suicide

Coba-coba merusak untuk kepuasan pribadi.

Jenis Hacker (Kemampuan)

1. 👑 Elite

Kasta tertinggi, paham detail sistem komputer dan jaringan.

2. 🥈 Semi Elite

Lebih muda dari Elite, menggunakan software bantuan (exploit).

3. 📚 Developed Kiddie

Masih bersekolah, mencoba keamanan jaringan sekolah.

4. 📜 Script Kiddie

Kemampuan dan pemahaman minim terhadap sistem.

5. 🔧 Lamer

Belum bisa disebut hacker, hanya menggunakan program jadi.

Social Engineering (1)

🎭 Definisi

Teknik pengumpulan informasi memanfaatkan celah psikologi target.


🎪 Metode

Langsung meminta informasi atau menciptakan situasi palsu.

🛡️ Pencegahan

  • Peraturan tertulis
  • Password kuat
  • Batasan hak akses

Social Engineering (Proses)

1. 🔍 Information Gathering

Mengumpulkan informasi dari target.

2. 🤝 Development of Relationship

Membangun kepercayaan dengan target.

3. ⚡ Exploitation

Mengeksploitasi kepercayaan untuk informasi.

4. 🎯 Execution

Melaksanakan tujuan penyerang.

Backdoor (1)

🚪 Definisi

Software akses sistem tanpa proses autentikasi.


⚡ Fungsi

Mengambil hak akses website dan server.

🔧 Penanggulangan

  • Bersihkan web hosting
  • Cari file backdoor

Backdoor (2)

📂 Jenis

  • Web Shell Backdoor: Pintu masuk melalui web shell untuk eksekusi program.
  • System Backdoor: Jenis paling populer, memberikan fleksibilitas dan stabilitas.

⚠️ Bahaya

  • Pencurian data
  • Pembajakan server
  • Serangan DDoS

Malware: Jenis Berbahaya

🦠 Virus

Kode menular menempel pada software.

🐛 Worms

Software mandiri bereplikasi.

🐴 Trojan Horse

Menyamar sebagai program berguna.

🕵️ Rootkit

Software tersembunyi di balik layar.

🔒 Ransomware

Mengunci sistem, menuntut tebusan.

⌨️ Keylogger

Merekam semua aktivitas keyboard.

Malware: Lainnya & Pencegahan

📋 Jenis Lainnya

  • Adware: Program berbasis iklan.
  • Spyware: Memata-matai aktivitas pengguna.
  • Rogue Security: Software keamanan palsu.
  • Browser Hijacker: Mengubah pengaturan browser.

🛡️ Pencegahan

  • Instalasi software anti-malware/virus
  • Aktifkan fungsi Firewall
  • Lakukan scan rutin secara menyeluruh
  • Bersihkan USB/Flash Drive yang terinfeksi

Kesimpulan

📜 Hukum Siber Penting

Mencegah kejahatan dunia maya dan melindungi masyarakat dari dampak kerugian.

🔥 Firewall Vital

Sistem keamanan jaringan yang melindungi sumber daya dari ancaman eksternal.

⚖️ Hacking Dual Peran

Hacking untuk kejahatan vs Ethical Hacking untuk pengujian keamanan.
