Fungsi Sigmoid

Mengubah nilai menjadi rentang 0 sampai 1.

A = 1 / (1 + e-Z)

Berguna untuk klasifikasi probabilistik.

Forward Pass XOR

  1. Hitung Z1
  2. A1 = sigmoid(Z1)
  3. Hitung Z2
  4. A2 = sigmoid(Z2)
  5. Hitung Error

Backpropagation

Proses memperbaiki bobot dari belakang ke depan.

  • 1. Forward pass
  • 2. Hitung error
  • 3. Update bobot output layer
  • 4. Update bobot hidden layer
  • 5. Ulangi
